## Project Overview 
This sample project initially shows the list of users in table format and by clicking on "Show Activity" the activities gets opened up in modal in table format for better view and easy to understand. On Clicking "Show in calendar" the calender view gets open, focusing on the last activity week.
The project is built in such a way that, if your server settings allow, you can directly paste the url to open up the calendar view of user having id <"XYZ">
